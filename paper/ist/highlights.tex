\noindent\textbf{Highlights}
\begin{itemize}[leftmargin=1.5em,nosep]
  \item Nine Node.js data-access layers benchmarked on PostgreSQL and MySQL via Express
  \item Throughput and tail latency reported jointly, unlike single-metric vendor tests
  \item Deep nested fetches expose up to 6.2x overhead between native drivers and ORMs
  \item Layer ranking is stable across engines; MySQL writes are InnoDB-commit bound
  \item Open reproducible harness with an automated cross-layer correctness check
\end{itemize}
